# URL SHORTENER

-takes in *valid url* and returns a **shortened url** 
-also keep track of *total clicks on url*

## Routes

### POST/URL 
-Generates a new short URL and returns the shortened URL int the format example.com/random - id.

### GET/:id
-Redirects the user to original URL

***GET/URl/analytics/:id***
-return the no of clicks
